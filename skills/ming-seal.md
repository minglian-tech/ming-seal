# 铭印 MingSeal - BSV司法存证

## 简介
铭印是OpenClaw龙虾的BSV区块链司法存证技能，将AI Agent的对话、决策哈希锚定到BSV链上。

## 触发词
- 存证、铭印、盖章
- 验证 TxID
- 证书 编号

## 功能说明
1. **手动存证**: 用户发送"存证"触发存证流程
2. **自动存证**: 配置开启后对话结束自动存证
3. **Merkle聚合**: 批量记录聚合降低成本
4. **BSV锚定**: OP_RETURN写入区块链
5. **验证证书**: 查询链上存证状态

## 使用示例
```
用户: 存证
插件: ✅ 铭印存证 [#MS20260503-001]
      📦 1条记录 | 🔗 abc123...(待上链)
```
